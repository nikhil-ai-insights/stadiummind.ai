# 🏟️ StadiumMind AI

> **Smarter Stadiums. Better Fan Experience.**

An AI-powered smart stadium platform built for the **Google PromptWars – FIFA World Cup 2026 Challenge**.

StadiumMind AI leverages **Google Gemini** to transform how fans, organizers, volunteers, and venue staff experience and manage large sporting events. The platform provides intelligent real-time assistance, operational insights, accessibility support, multilingual communication, crowd intelligence, and emergency decision support through Generative AI.

---

# 🌍 Overview

Managing millions of fans across multiple FIFA World Cup venues requires intelligent systems that can improve navigation, reduce congestion, enhance accessibility, and help organizers make faster decisions.

StadiumMind AI acts as an AI-powered operations and fan experience platform capable of assisting every stakeholder inside a stadium ecosystem.

Whether a fan is trying to find their seat, a volunteer needs emergency guidance, or organizers want live operational intelligence, StadiumMind AI delivers contextual AI-powered recommendations in real time.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/stadiummind-ai.git
```

Navigate into the project:

```bash
cd stadiummind-ai
```

Install dependencies:

```bash
npm install
```

Create an environment file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
